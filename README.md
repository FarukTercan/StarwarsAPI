

# StarWarsAPI

> StarWarsAPI is a web app that uses the Star wars API (SWAPI) to display the list of films about the StarWars universe.
> 



## Starting Repo - The Complete Guide
Clone down this repository. 

Installation:

`npm install`

To Run Test Suite:

`npm test`

To Start Server:

`npm start`

To Visit App:

`localhost:3000`
